Our Commitment to Privacy

Tobagin Apps is committed to protecting your privacy. This privacy policy explains how we collect, use, and protect information when you visit our website.

Privacy-First Analytics

We use a custom-built, privacy-first analytics system to understand how visitors use our site and improve the experience. Our approach is designed with privacy by default:

What We Collect

  • Anonymous usage data: Which apps you view, search terms (anonymized), and filters you use
  • Technical information: Browser type (without version), screen size, language preference
  • Performance data: Page load times and user experience metrics

What We Don't Collect

  • ❌ No cookies or persistent tracking identifiers
  • ❌ No personal information (name, email, address)
  • ❌ No IP addresses or precise location data
  • ❌ No cross-site tracking or third-party data sharing
  • ❌ No behavioral profiles or advertising data

Privacy Safeguards

  • Do Not Track compliance: We automatically respect your browser's Do Not Track setting
  • Data anonymization: All data is anonymized before collection
  • Time bucketing: Timestamps are rounded to 5-minute intervals
  • Limited retention: Data is aggregated and individual events are discarded
  • No external tracking: All analytics run on our own infrastructure

Your Rights and Controls

You have complete control over your privacy:

Opt-Out Options

  • Browser setting: Enable "Do Not Track" in your browser
  • Site setting: Use the opt-out controls below
  • Local storage: We only use localStorage to remember your opt-out preference

GDPR Compliance

Our privacy-first approach ensures compliance with the General Data Protection Regulation (GDPR) and other privacy laws:

  • Lawful basis: Legitimate interest in improving our service
  • Data minimization: We collect only what's necessary
  • Purpose limitation: Data used only for site improvement
  • Right to object: Easy opt-out at any time
  • No consent required: Anonymous data with clear opt-out

External Services

Our website may link to external services with their own privacy policies:

  • Flathub: App installation links go to flathub.org
  • GitHub: Source code links go to github.com
  • CDNs: Web Vitals library loaded from unpkg.com

We recommend reviewing the privacy policies of these services. We do not control or share data with these external services.

Data Security

We implement appropriate security measures to protect collected data:

  • Data is processed client-side before aggregation
  • No persistent storage of individual user data
  • Regular security reviews of our analytics implementation
  • Minimal data collection reduces exposure risk

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date. Continued use of the site after changes constitutes acceptance of the updated policy.

Contact Us

If you have questions about this privacy policy or our data practices, please contact us:

  • GitHub: Create an issue
  • Email: Through GitHub contact methods

Technical Details

For developers and privacy advocates

Implementation Details

  • Custom JavaScript analytics (no third-party trackers)
  • Client-side data anonymization before any processing
  • No server-side logging of individual requests
  • Progressive Web App with offline-first caching
  • Open source - view our implementation on GitHub

Data Processing

  • Events aggregated in 5-minute time buckets
  • User agents stripped of version information
  • Referrers reduced to domain-only
  • Search queries anonymized (numbers → N, special chars removed)
  • Maximum 50 events stored per session in memory

Privacy by Design

  • No unique identifiers (session IDs are non-persistent)
  • Do Not Track header automatically respected
  • Opt-out preference stored locally only
  • No cross-domain or cross-session tracking
  • GDPR compliant without requiring consent banners
